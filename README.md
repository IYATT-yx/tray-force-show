# tray-force-show 强制显示托盘图标工具

2025/12/16  
我已经使用 Windows 11 有 4 年了，记得 2021 年刚推送更新的时候我就升级了。升级后最无法接受的就是资源管理器中右键折叠菜单和托盘图标不能一键设置全部显示，右键菜单倒可以按住 Shift 点击右键就可以显示完整菜单，最让人困扰的是每次新增了带托盘的软件，托盘图标默认是隐藏的，需要手动去设置显示出来（我是习惯显示全部托盘图标的，有什么后台软件在运行就一目了然）。  
用了 4 年后的我突发奇想写一个小工具实现监视托盘图标，有新增的就强制显示出来。上周就初步有这个想法，但是下班后后我不太想动脑了，希望可以好好玩，就拖了好久，今晚上一下搞定。目前简单测试可以按预期运行，如果发现有问题就再修复更新吧。  

运行后会在托盘显示本工具图标，可以右键勾选开机自启动

# 测试环境

* Windows 11 25H2  
* Python 3.13.11  

打包的可执行文件也基于这个环境，不支持 Windows 8 以下运行，当然实际也只有 Windows 11 需要使用这个工具，Windows 11 以下可以在控制面板一键设置显示全部托盘图标的。
